# Privacy Policy

**Last Updated: December 29, 2025**

## 1. Introduction

This Privacy Policy explains how Discora ("the Bot", "we", "us", or "our") collects, uses, stores, and protects your information when you use our Discord bot service. We are committed to protecting your privacy and complying with applicable data protection laws.

## 2. Information We Collect

### 2.1 Automatically Collected Information

When you use the Bot, we automatically collect:

**Discord Server Information:**
- Server ID (guild ID)
- Server name
- Server member count
- Server owner ID
- Channel IDs where the Bot is used

**Discord User Information:**
- User ID
- Username and discriminator
- Avatar URL
- Roles in the server
- Join/leave timestamps

**Bot Usage Data:**
- Commands executed
- Command parameters (excluding sensitive content)
- Module enable/disable actions
- Configuration changes
- Error logs and debug information
- Timestamps of interactions

### 2.2 Information You Provide

**Server Configuration:**
- Welcome messages and settings
- Moderation configurations
- Auto-role settings
- Custom command responses
- Module preferences per server

**Account Information (Dashboard):**
- Discord OAuth credentials (managed by Discord)
- Email address (if provided via Discord)
- Subscription plan selection
- Payment information (processed by third-party payment providers)

### 2.3 Information We Do NOT Collect

We do not collect:
- Message content (except when directly mentioned in commands)
- Private/Direct Messages
- Voice chat content
- Files or attachments
- Credit card numbers (handled by payment processors)
- Passwords (authentication via Discord OAuth)

## 7. Your Rights and Choices

### 7.1 Access and Control

You have the right to:
- **Access**: Request a copy of your data
- **Correction**: Update incorrect information via the dashboard
- **Deletion**: Request deletion of your data (see Section 7.2)

1. Remove the Bot from your server (deletes server configurations)
